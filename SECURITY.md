# Security Policy

## Supported Versions

Please see [Releases](https://github.com/dapplion/vibehouse/releases/).

## Reporting a Vulnerability

Please report vulnerabilities via [GitHub Issues](https://github.com/dapplion/vibehouse/issues) (private disclosure preferred — use GitHub's security advisory feature).

**Please do not file a public ticket** mentioning the vulnerability, as doing so could increase the likelihood of the vulnerability being used before a fix has been created, released and installed on the network.
